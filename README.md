# Order tracking

A sample project to explore integration testing for the Nextjs app.

Used for training purpose within [Better Engineering Community](https://www.withnik.com/community) founded by Nik Sumeiko.

## Prerequisites

- Node.js 22.18+
- npm

## Getting started

```bash
npm install
npm run test:integration
```

This will open Playwright UI with a test asserting user journeys behind the app.

## Run the app

Spin up the API along with the app:

```bash
npm run api
``` 
```bash
npm run dev
```

Or both in one command:

```bash
npm start
```
